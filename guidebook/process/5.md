# 流程5-迈入电气时代

# 发电

制造<ItemLink id="anvilcraft:charge_collector" />，通过本模组的特殊方法发电

# 输电

- 制造<ItemLink id="anvilcraft:transmission_pole" />，它们可以创建巨大的电网
- 电网可以将附近所有电力设施连接在一起
- 将<ItemLink id="anvilcraft:anvil_hammer" />戴在头上，就可以看到电网

# 用电

- 学习使用<ItemLink id="anvilcraft:heater" />进行高温熔炼
- 使用<ItemLink id="anvilcraft:charger" />制作<ItemLink id="anvilcraft:magnet_ingot" />
- 使用<ItemLink id="anvilcraft:ionocraft_backpack" />尽情飞翔
- 使用<ItemLink id="anvilcraft:induction_light" />催生农作物
