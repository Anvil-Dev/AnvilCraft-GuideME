# 流程1-开局

> 蓝色字体可以点击，前往关联页面

# 前期必备工具
- [紫水晶工具](../prop/amethyst_tools.md)自带附魔，且十分耐用，建议打造一款
- 如果有多余的经验，可以在铁砧内合并相同的紫水晶工具，获得更高等级的附魔

# 紫水晶探查
- <ItemLink id="anvilcraft:geode" />可以定位紫水晶洞穴，快速获得足够的紫水晶
- 可以将一部分<ItemLink id="minecraft:budding_amethyst" />挖掘，获得<ItemLink id="anvilcraft:geode" />，之后有大用

# 建造刷铁机
- 用铁砧砸击铁傀儡，如符合模组设定的条件，可以产生铁锭
- 模组允许发射器使用铁锭修补铁傀儡
- 加以红石系统，可以制造远超原版效率的刷铁机
