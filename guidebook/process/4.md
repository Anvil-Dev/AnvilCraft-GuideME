# 流程4-迈入电气时代

# 发电

制造<ItemLink id="anvilcraft:charge_collector" />，通过本模组的特殊方法发电

# 输电

- 制造<ItemLink id="anvilcraft:transmission_pole" />，它们可以创建巨大的电网
- 电网可以将附近所有电力设施连接在一起
- 将<ItemLink id="anvilcraft:anvil_hammer" />戴在头上，就可以看到电网
