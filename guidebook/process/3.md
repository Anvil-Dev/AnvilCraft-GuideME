# 流程3-学习铁砧加工

> 蓝色字体可以点击，前往关联页面

# 制造<ItemLink id="anvilcraft:anvil_hammer" />

通过<ItemLink id="anvilcraft:anvil_hammer" />，可以快速拆卸本模组的大部分方块，值得合成

# 铁砧加工

- <ItemLink id="minecraft:anvil" />落在特定的方块上时，可以加工该方块
- <ItemLink id="minecraft:anvil" />落在特定的方块上时，会加工其中或其上的物品
- 使用<ItemLink id="anvilcraft:magnet_block" />吸引铁砧，再用红石信号使<ItemLink id="anvilcraft:magnet_block" />失去磁性，使铁砧下落

> 若没有磁铁块，手动拆除铁砧再将其放置或者设计红石机器也未尝不可
