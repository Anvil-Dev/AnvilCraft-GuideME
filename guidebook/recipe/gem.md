# 量产绿宝石

- 拥有<ItemLink id="anvilcraft:giant_anvil" />后，可以[制造刷怪笼](spawner.md)
- 获得卫道士刷怪笼
- 击杀卫道士获得<ItemLink id="minecraft:emerald" />

# 自动化

- 尽管卫道士只有被玩家击杀，才会掉落<ItemLink id="minecraft:emerald" />
- 我们仍可以通过<ItemLink id="anvilcraft:ember_anvil" />对其进行[铁砧抢夺](../feature/anvil_loot.md)，自动化生产<ItemLink id="minecraft:emerald" />
