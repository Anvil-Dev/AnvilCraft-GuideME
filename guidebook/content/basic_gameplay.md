# Basic Gameplay

The basic gameplay of AnvilCraft mainly includes changes caused by the falling anvil, which can transform blocks, process items, affect mobs, etc.
This mod adds some blocks to facilitate early automation, including block placer, chute, etc., which can be viewed in the *Basic More Device* entry.
