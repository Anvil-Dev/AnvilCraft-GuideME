# 共振器

- 万能工具，可以挖任何方块
- 不会完全损坏，类似于鞘翅，它会失去所有增益、功能等效果，攻击伤害变为0，附魔大部分失效，浮霜带来的[无情](../feature/properties.md#无情)也会失效
- 按住<Key id="key.anvilcraft.switch_resonate_mode"></Key>键呼出轮盘，轮盘上有五个选项
  - 默认选择Auto，加速破坏任意方块
  - 选择对应的工具类型时，只加速破坏对应的方块，但可以执行相应的*去皮*、*耕地*等操作
- 共振挖掘：对任意方块长按右键一小段时间立刻挖掘下来，专制坚硬方块 (<ItemLink id="minecraft:bedrock" />这类方块除外)
