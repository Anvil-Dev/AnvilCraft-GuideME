# 嬗变锻造

<Recipe id="anvilcraft:permutation_smithing_template"></Recipe>
保留物品的形态，转换其材质

- 消耗<ItemLink id="anvilcraft:royal_steel_ingot" />，转化钻石武器工具 和 皇家钢武器工具
- 消耗<ItemLink id="anvilcraft:ember_metal_ingot" />，转化下界合金武器工具 和 余烬金属武器工具
- 消耗<ItemLink id="anvilcraft:multiphase_matter" />，转化浮霜金属武器工具 和 余烬金属武器工具
- 消耗<ItemLink id="anvilcraft:multiphase_matter_block" />，转化浮霜铁砧、锻造台、砂轮 和 余烬铁砧、锻造台、砂轮
- 消耗<ItemLink id="anvilcraft:chromatic_stone" />，转化四种[宝石护符](../prop/amulet.md)

# 形变锻造

<Recipe id="anvilcraft:deformation_smithing_template"></Recipe>
保留物品的材质，转换其形态

消耗<ItemLink id="anvilcraft:frost_metal_ingot" />使以下工具形态之间可以相互转换：
- 剑，斧，镐，锄，锹
- 头盔，胸甲，腿甲，靴子
- 弓，弩
