# 功能

- 可以照明
- 对其使用不同物品右键可以设置功能模式，详细见下表

|                    对应物品                    | 工作范围     | 功能                      | 消耗电力(kW) |
|:------------------------------------------:|----------|-------------------------|:--------:|
|                     -                      | -        | -                       |    1     |
|    <ItemLink id="minecraft:redstone" />    | 5x5x5的方块 | 催熟作物，要求对象的方块光等级不小**10** |    16    |
| <ItemLink id="minecraft:glowstone_dust" /> | 3x3x3的区段 | 抑制**怪物**的生成             |    16    |
|  <ItemLink id="anvilcraft:void_matter" />  | 3x3x3的区段 | 抑制**动物**的生成             |    16    |

> 区段：**区块**的一截，16x16x16大小。按F3+G可查看，以蓝线切分。

# 特性

- 电力不足时无法工作
