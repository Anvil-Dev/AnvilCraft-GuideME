# 生产
铁砧落入末地门有概率变成<ItemLink id="anvilcraft:spectral_anvil" />:
- <ItemLink id="minecraft:damaged_anvil" />：1%
- <ItemLink id="minecraft:chipped_anvil" />：2%
- <ItemLink id="minecraft:anvil" />：3%
- <ItemLink id="anvilcraft:royal_anvil" />：50%
- <ItemLink id="anvilcraft:ember_anvil" />：100%
- <ItemLink id="anvilcraft:transcendence_anvil" />：100%

# 特性
- 不会受到重力影响
- 可被活塞推动
- 使用时，不会损坏
- 当头顶的<ItemLink id="anvilcraft:magnet_block" />消失或磁性消失时，会向下掉落一个**虚影**
- **虚影**无论从多高砸到一个物体，都相当于从2格高下落的铁砧

# 与铁砧的不同之处
- 无法进行[铁砧抢劫](../feature/anvil_loot.md)
- 砸击<ItemLink id="anvilcraft:piezoelectric_crystal" />产生的电荷也会偏少
