# 功能

- <ItemLink id="anvilcraft:block_placer" />受到红石信号或被铁砧砸时，放置方块
- 从其背后的容器方块、实体库存、掉落物中取用物品

> eg: <ItemLink id="minecraft:chest" /> <ItemLink id="anvilcraft:chute" /> <ItemLink id="minecraft:oak_chest_boat" />

- 受到红石信号时，于面前放置方块
- 被铁砧砸时，根据铁砧下落高度 n ，间隔 n 格放置方块

# 特性

- 可以被活塞推拉
